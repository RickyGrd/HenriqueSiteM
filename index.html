Projetos

Projeto 1

1. Sistema de Gerenciamento de Tarefas Descrição: Desenvolvi um sistema de gerenciamento de tarefas utilizando JavaScript, Node.js e MySQL. O sistema permite que os usuários criem, editem e excluam tarefas, além de marcar tarefas como concluídas. Inclui autenticação de usuário e uma interface intuitiva para facilitar a gestão das atividades diárias.

Projeto 2

Aplicativo de Gestão Hoteleira Descrição: Como parte do meu curso de Análise e Desenvolvimento de Sistemas, desenvolvi um aplicativo de gestão hoteleira para um hotel fictício. Este aplicativo foi projetado para simplificar a administração do hotel, oferecendo funcionalidades como reservas de quartos, check-in/check-out, gestão de hóspedes e geração de relatórios financeiros. O sistema possui uma interface amigável e responsiva, permitindo que os funcionários do hotel possam acessar e gerenciar as informações de qualquer dispositivo.

Projeto 3

Fluxograma para Escola de Cursos de Idiomas Descrição: Também como parte do meu curso de Análise e Desenvolvimento de Sistemas, desenvolvi um projeto de fluxograma para otimizar os processos administrativos de uma escola de cursos de idiomas. O objetivo do projeto foi mapear e melhorar o fluxo de informações e tarefas dentro da instituição, garantindo maior eficiência e clareza nas operações diárias..
